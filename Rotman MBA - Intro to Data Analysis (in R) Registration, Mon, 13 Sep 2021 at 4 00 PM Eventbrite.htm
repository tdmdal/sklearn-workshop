About this event

Rotman MBA

Intro to Data Analysis (in R)

This R workshop will introduce students to the basics of the R coding language. It allows students plenty of hands-on programming using R. The practical component includes projects that teach students to load data, assemble and disassemble data objects, navigate R’s environment system, write their own functions, and use all of R’s programming tools, such as 'if else' statements and for 'loops'. Furthermore, the workshop will teach not just how to program with R, but how to use the programming skills to support your work as a data scientist.

Students should download and install R from https://cran.r-project.org/

Topics included are:

  • The very basics of R
  • Packages and help pages
  • R objects
  • R notation
  • Modifying values and programs
  • Functions
  • Plotting
  • Data frames
  • Programs
